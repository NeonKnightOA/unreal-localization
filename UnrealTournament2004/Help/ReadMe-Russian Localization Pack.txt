Version 2.6.

Translation of Unreal Tournament 2004. In addition to the Russification of the base game, it also contains translations of some mods. Russifier allows you to write in chat in Russian, while not replacing standard fonts (does not cause conflicts when connecting to servers). All names (weapons, maps, vehicles, etc.) are in English.

To install, unzip the archive to the root folder of the game, confirming the replacement of files.

The author of the original translation is Rasiel. Updated Cobalt.
